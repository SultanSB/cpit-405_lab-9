# Recipe Search App

A simple React application for searching and viewing recipes.

## Features

- **Search Recipes:** Find recipes by name.
- **Recipe Details:** View detailed information about specific recipes.

## Technologies Used

- **React:** UI library for building the interface.
- **Vite:** Fast build tool and development server.
- **React Router:** For navigation between pages.
- **CSS:** Standard styling.


## Project Structure

- `src/components`: Contains reusable components like `Search` and `RecipeDetails`.
- `src/App.jsx`: Main application component with routing logic.
- `src/main.jsx`: Entry point of the application.
